表格

1.表格标签

用到的标签有thead(表头),tbody(表躯干),tfoot(表注),th(可用于表头，也可以用于每一行开头的元素),tr(表格里的每一行),td(表格里的每一个单元格),如果就单纯用以上元素做表格，那么做成的表格每个单元格是隔开的,要想单元格合并在一起，就需要给table加上border-collapse: collapse;

单元格合并

如果要将多个单元格合并，就给最左上的单元格添加属性，如果是同一行的单元格合并，就给最左边的单元格添加colspan="",并在""内注明要合并的单元格数量，如果是同一列的单元格合并，就给最上方的单元格添加rowspan="",并在""内注明要合并的单元格的数量

表单

用于收集用户信息

1.表单域

用于收集用户信息，并向后台传送，格式如下form /form

2.表单控件

  • button
    • 定义可点击按钮(多数情况下，用于通过JavaScript启动脚本)
  • checkbox
    • 定义复选框
  • file
    • 定义输入字段和"浏览"按钮，供文件上传
  • hidden
    • 定义隐藏的输入字段
  • image
    • 定义图像形式的提交按钮
  • password
    • 定义密码字段，该字段中的字符被掩盖
  • radio
    • 定义单选按钮
  • reset
    • 定义重置按钮，重置按钮会清除表单内的所有数据
  • submit
    • 定义提交按钮，提交按钮会把表单form里的数据发送到后台服务器
  • text
    • 定义单行的输入字段，用户可在其中输入文本，默认宽度为20个字符
