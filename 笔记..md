# css之选择器

## 1.标签选择器

> 适用于修改某一个标签内的所有文本的样式，格式如下
>
    p {
        color: green;
    }

## 2.类选择器

> 适用于一个样式，多个不同标签内的元素要使用。格式如下
>
    .red {
        color: red;
    }

## 3.多类名的使用方法

> 适用于一个元素要使用多个的单个样式，可以用多类名,格式如下

    .red {}
    .font-family {}
    .font-size {}
      <p class="red font-size font-family">
        多类名     
      </p>

## 4.id选择器

> id选择器只能调用一次，格式如下
>
       #red {}
       <p id="red">id选择器</p>

## 5.通配符选择器

> 适用于给页面内所有的元素修改格式，格式如下
>
       * {想要修改的属性}
